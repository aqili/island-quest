# 🏝️ Island Quest

A 3D browser-based educational game for kids aged 6–12. Explore Math Island and Language Island, solve puzzles in each castle room, and earn crowns!

## How to Run Locally

Just open `index.html` in your browser — no install needed!

## Deploy to Vercel

1. Push this repo to GitHub
2. Import it on [vercel.com](https://vercel.com)
3. No configuration needed — deploy!
